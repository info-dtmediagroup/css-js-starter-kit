Ein kleines Plugin um schnell und einfach CSS & Javascript Snippets in den Header bzw. Footer hinzuzufügen.

== Description ==

Über das Plugin kann man über ein übersichtliches Backend verschiedene CSS & JS Bibliotheken oder andere Snippets einfügen.
Die Features können nach Bedarf erweitert werden. Updates laufen über ein Github Repository

== Installation ==

1. Upload 'css-js-starter-kit'
2. Im Backend aktivieren
